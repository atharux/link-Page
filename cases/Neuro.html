Psychological Framework

The application leverages Octalysis Framework and Self-Determination Theory (SDT) to transform "tasks" into "quests". We focus primarily on White Hat triggers to foster long-term retention and well-being.

Designing for Flow State

The app's AI balances task difficulty with user skill. If the AI detects a user completing tasks too quickly (Boredom), it suggests "High Complexity" tasks. If velocity drops (Anxiety), it recommends "Quick Wins".

  • High Challenge + Low Skill = Anxiety
  • Low Challenge + High Skill = Boredom
  • Balanced = Flow State (The Zone)

Deterministic AI Advisory

We don't just rely on LLMs (which can hallucinate). We use a Weighted Scoring Model (WSM) to deterministically calculate task priority. The AI acts as a filter, sorting the backlog so users focus on high-impact learning.

// The Priority Formula
Score = (Impact * Urgency) / Effort
• Impact (1-10): Career relevance
• Urgency (1-5): Due date proximity
• Effort (1-10): Time estimate
