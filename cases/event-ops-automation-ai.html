Event Ops Automation with AI

Overview

Event planners manage complex operational workflows involving budgets, vendors, logistics, and risk—often across disconnected tools and manual processes.

I designed a Python-based reference implementation to validate a structured system for automating event operations. The system separates deterministic rules (costs, budget constraints, risk detection) from an AI advisory layer that suggests optimizations without modifying facts.

The reference implementation confirms the architecture works and integrates safely with an existing event planning UI, without requiring any UI changes.

My Role

  • Defined system architecture and workflows
  • Modeled business rules and deterministic logic for event operations
  • Designed AI orchestration patterns and boundaries
  • Validated system behavior through a Python reference implementation, assisted by an AI coding tool
  • Authored documentation and case study for portfolio integration

Note: The Python reference implementation was co-developed with an AI coding assistant. My focus was on architecture, rules, validation, and portfolio-level design, not production-level coding.

System Architecture

The system has two layers:

  • Deterministic Operations Layer: Calculates costs using explicit rules, evaluates budget constraints, detects risks, and generates suggested next actions.
  • AI Orchestration Layer: Converts deterministic outputs into AI-safe context, suggests optimizations without modifying system facts, and validates advisory recommendations against business rules.

Methodology & Tooling

  • Python 3.12 — reference implementation, not production backend
  • JSON-first architecture — separates data, rules, and AI prompts
  • AI-assisted coding — used AI to generate code scaffolding and logic, while I directed architecture and verified behavior
  • Validation — tested outputs with realistic input events to confirm expected operational behavior
  • No UI changes — system designed to integrate seamlessly with the existing event planner interface

Technical Highlights

  • Rule-based cost estimation
  • Budget and risk evaluation
  • AI advisory context generation
  • Deterministic vs probabilistic separation
  • System validation via Python reference implementation
